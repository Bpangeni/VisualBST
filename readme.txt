
# Visual BST using SFML and  C++

A binary search tree, also called an ordered or sorted binary tree, is a rooted binary tree data structure whose internal nodes each store a key greater than all the keys in the nodeâ€™s left subtree and less than those in its right subtree.


## Run locally on PC 

Clone the project using the Git command 

```bash
  git clone https://github.com/Bpangeni/VisualBST.git
```

Change your current directory to Project directory
```bash
  cd VisualBST
```
Compile 
```bash
  mingw32-make
```
Run the program
```bash
  .\main.exe  
```
